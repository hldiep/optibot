# OptiBot Knowledge Base Sync

A Python application that automatically scrapes OptiSigns Help Center articles, converts them to Markdown, and prepares them for AI-powered knowledge retrieval.

## Features

- Scrape articles from OptiSigns Help Center
- Convert HTML to Markdown
- Save articles in a local `docs/` folder
- Detect new or updated articles
- Upload knowledge to AI service (OpenAI/Gemini)
- Ready for scheduled daily synchronization

## Tech Stack

- Python 3.11
- Requests
- BeautifulSoup4
- Markdownify
- python-dotenv
- Google Gemini API
